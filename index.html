attr()

A css function that allows you to insert a value within CSS taken from an attribute of an HTML element


						#newDiv {
							background: gray;
							padding: 10px;
						  }

						#newDiv::after {
							content: ' Some Text';
						  }
					

						#newDiv {
							background: gray;
							padding: 10px;
						  }

						#newDiv::after {
							content: attr(id);
						  }
					

						#newDiv::after {
							content: " The ID attribute for this div is '" attr(id) "'...Awesome!";
							background: green;
						  }

clamp()

A CSS function that takes three value parameters (minimum, preferred, and maximum) and makes sure the value always stays withing this range.

CSS magic for responsive design


                        body {
                            font-family: basic-sans, sans-serif;
                            min-height: 100vh;
                            display: flex;
                            justify-content: center;
                            align-items: center;
                            font-size: 1rem;
                            line-height: 1.6;
                        }

                        .content {
                            background: lightblue;
                            padding: 3em;
                            width: 70%;
                            box-shadow: 0 0 2em rgba(0, 0, 0, 0.199);
                        }

                        .title {
                            color: rgb(255, 118, 64);
                        }
					

                        .title {
                            color: rgb(255, 118, 64);
                            font-size: clamp(2rem, 5vw, 3rem);
                        }
